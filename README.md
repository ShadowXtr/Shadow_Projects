# Shadow_Projects
Projects Made

1. Tic Tac Toe Game: 
Made using Python3. A standard Tic Tac Toe game. This game has been made using a Player vs Player model.

2. Three Digit Guess Game:
Made using Python3. Computer will randomly generate a 3 digit number and the player has to keep guessing the three digit number to end the game. Each number guessed and wrongly guessed makes the computer give a clue back as to how close the digits guessed are with the actual number. It also tracks the number of turns which the player took to guess the right number.

3. Random Credit Card Number Generator:
Made using Python3. This program generates 16-digit Credit Card Numbers satisfied using the Luhn's algorithm whose contents can be copied which can be later used to test credit card logins in various websites while designing.
